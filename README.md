# 🏛️ 翡翠书斋v0.0.1beta

> **翡翠书斋**——一座被遗忘的贤者殿堂，六重领域封印着永恒智慧。每一次开启，密码皆随机而生，探索永不重复。


## 📖 游戏简介

《翡翠书斋》是一款**纯浏览器解谜游戏**，无需安装，打开即可游玩。你将以探索者的身份，穿越 **门厅 → 地窖 → 书房 → 花园 → 塔楼 → 圣殿** 六重领域，收集道具、破解密码、解锁故事，最终揭开贤者 **艾德蒙·格雷** 留下的终极秘密。

### ✨ 核心特色

- **动态密码系统**：所有三位数密码（书架、花园门、书柜、宝箱等）均在游戏开局时**随机生成**（100~999），每次重开都带来全新的谜题顺序，重复可玩性极高。
- **渐进式叙事**：随着游戏进程，**12 篇编年史**逐步解锁，为你揭示翡翠书斋的古老传说与格雷的哲思。
- **沉浸式交互**：点击物品、使用道具、输入密码，所有操作即时反馈，沉浸于暗色复古书斋氛围。

## 🎮 玩法与机制

### 场景推进
游戏包含 **6 个主场景**，解锁顺序固定，必须依次完成机关：

| 场景 | 解锁条件 |
|------|----------|
| 🏛️ 门厅 | 初始开放 |
| 🪦 地窖 | 在门厅将太阳徽章嵌入石像 |
| 📚 书房 | 在地窖用书房钥匙打开铁门 |
| 🌿 花园 | 在书房用银钥匙打开书桌抽屉 |
| 🗼 塔楼 | 在花园输入正确的花园门密码 |
| ✨ 圣殿 | 在塔楼集齐三件圣物并打开宝箱 |

### 道具系统
- 点击场景中的物品（如肖像画、落地钟、石棺、书架等）获取**关键道具**。
- 道具出现在右侧物品栏中，可用于后续交互（例如用“铜钥匙”打开石棺）。
- 部分道具是**组合型**的（如三件圣物合成符文钥匙），需要集齐后才能使用。

### 密码系统（随机生成）
游戏共包含 **9 个密码点位**，每次开局随机生成（100~999 的三位数）：

| 密码点位 | 线索来源 |
|----------|----------|
| 书架密码 | 擦亮的铜镜 |
| 花园门密码 | 壁炉纸条 / 书桌羊皮纸 |
| 书柜密码 | 园丁像纸条 |
| 宝箱密码 | 望远镜观察星座 |
| 火炬暗格密码 | 点燃地窖火炬后墙上显示 |
| 书房烛台密码 | 点燃书桌烛台后墙上显示 |
| 羽毛笔密码 | 蘸墨后书写显示 |
| 窗户密码 | 推开窗户后窗框显示 |
| 塔楼烛台密码 | 点燃塔楼烛台后墙上显示 |

所有密码线索均**动态显示**在游戏内，你无需记忆固定答案，只需细心观察环境。

### 故事系统
点击右上角「📖 故事」按钮，可查看已解锁的剧情片段。随着游戏进程，12 篇编年史将逐步呈现，为你揭开艾德蒙·格雷与翡翠书斋的传奇往事。

## 🏛️ 背景故事（节选）

贤者 **艾德蒙·格雷** 毕生钻研星象、元素与古老符文，他将毕生智慧封印于翡翠书斋的六重领域之中。书斋由他的家族世代守护，但格雷留下遗嘱：**唯有集齐六幕之钥的探索者，才能继承“永恒智慧”。**

你，作为误入书斋的旅人，将面临一场跨越光与影、月与日、自然与星空的考验。每一件物品、每一句铭文、每一道密码，都在诉说着格雷的哲思。当你最终站上圣殿祭坛，你将明白——智慧并非答案，而是探寻答案的旅程。
